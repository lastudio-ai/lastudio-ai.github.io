# LA Studio Model Catalog

Static GitHub Pages catalog for LA Studio curated speech models.

## Local preview

Run from this directory:

```powershell
C:\Users\daidu\miniconda3\python.exe -m http.server 4173 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4173/
```

## Data

The page loads `data/catalog.json`. Refresh that file from the root project `data/catalog.json` whenever the generated LA Studio catalog changes.
